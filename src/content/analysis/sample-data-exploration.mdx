## How It Works

Analysis articles on Hartzog.ai are written in **MDX** — Markdown with embedded React components. This means any article can include:

- **Interactive charts** (line, bar, and more) powered by Recharts
- **Sortable, filterable tables** for exploring datasets
- **Custom React components** for specialized visualizations

New analysis articles are simply `.mdx` files dropped into the content folder. They are automatically built and deployed when pushed to the repository.
